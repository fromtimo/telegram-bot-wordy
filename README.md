# Telegram Bot "Wordy"

Telegram-бот для изучения английского языка с упором на быстрый UX.

## Основные возможности
- FSM-логика (aiogram 3)
- Интерактивный интерфейс (ReplyKeyboardMarkup)
- Система XP и уровней
- Работа с базой данных (SQLite, SQLAlchemy)
- Поведенческий UX: одно активное сообщение в чате

## Стек
- Python
- aiogram 3
- SQLAlchemy
- SQLite

## Статус
Проект находится в стадии MVP. Запускался локально для разработки и тестирования.
